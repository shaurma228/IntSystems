# Диагностика диабета (ML проект)

## Описание проекта

Проект представляет собой систему машинного обучения, которая определяет наличие диабета у пациента на основе
медицинских показателей.

Модель принимает данные пациента и возвращает прогноз:  
0 — диабет отсутствует  
1 — диабет вероятен

---

## Цель проекта

Разработать и протестировать модель машинного обучения, которая:

- обучается на медицинских данных
- выполняет предсказания
- работает через веб-интерфейс
- проходит автоматическое тестирование

---

## Датасет

Используется **Pima Indians Diabetes Dataset**

### Признаки:

- Pregnancies
- Glucose
- BloodPressure
- SkinThickness
- Insulin
- BMI
- DiabetesPedigreeFunction
- Age

### Целевая переменная:

- Outcome (0 / 1)

---

## Структура проекта

- `src/`: Исходный код модулей
    - `data.py`: Загрузка и очистка данных
    - `model.py`: Архитектура моделей и подбор параметров
- `config.yaml`: Конфигурация проекта (пути, гиперпараметры)
- `train.py`: Скрипт для обучения моделей и выбора лучшей
- `app.py`: Веб-интерфейс на Gradio
- `tests/`: Автоматические тесты

---

## Особенности

1. **Структура проекта**: Логика разделена на модули, используется файл конфигурации.
2. **Очистка данных**: Нулевые значения в критических колонках (глюкоза, ИМТ и др.) обрабатываются как отсутствующие и
   заполняются медианой.
3. **Ансамбли моделей**: Logistic Regression, Random Forest и Gradient Boosting.
4. **Тюнинг гиперпараметров**: Используется `GridSearchCV` для поиска оптимальных настроек.
5. **UI**:
    - Отображение вероятности прогноза.
    - Визуализация важности признаков через `gr.BarPlot`.
    - Валидация входных данных.
6. **Расширенное тестирование**: Тесты на обработку данных и граничные случаи.
