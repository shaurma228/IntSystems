# Диагностика диабета (ML проект)

## Описание проекта

Проект представляет собой систему машинного обучения, которая определяет наличие диабета у пациента на основе медицинских показателей.

Модель принимает данные пациента и возвращает прогноз:  
0 — диабет отсутствует  
1 — диабет вероятен

---

## Цель проекта

Разработать и протестировать модель машинного обучения, которая:

- обучается на медицинских данных
- выполняет предсказания
- работает через веб-интерфейс
- проходит автоматическое тестирование

---

## Датасет

Используется **Pima Indians Diabetes Dataset**

### Признаки:
- Pregnancies
- Glucose
- BloodPressure
- SkinThickness
- Insulin
- BMI
- DiabetesPedigreeFunction
- Age

### Целевая переменная:
- Outcome (0 / 1)

---

## Используемые технологии

- Python
- Pandas
- NumPy
- Scikit-learn
- Gradio
- Pytest

---

## Модель

Использованы алгоритмы:
- Logistic Regression
- Decision Tree (для сравнения)

Лучшая модель: Logistic Regression  
Средняя точность: ~0.75–0.76
